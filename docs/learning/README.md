## Method

- Window: 2025-06-01 to 2026-05-31
- Source: `stock_event_paper_trades` where `status='closed' AND horizon_days=7`
- Bankroll: $5,000, $$1,000 per position, max 5 concurrent
- Slippage: already netted in `realized_return` (10 bps round-trip)
- Horizon: 7d — chosen because h7d shows highest profit_factor
  for the dominant 8K/news rule_keys per the sector audit, and matches
  a realistic 'alert + 1 week' trader holding period.

**This is not advice and not a backtest of a real strategy.** It's a
counterfactual: 'what if the $5K shadow portfolio had been live during
this entire window?'
